Bull--itPoint: Flexible Presentation Software

Bull--itPoint (pronounced like bullet point) is a presentation software that creates slides in realtime using speech-to-text software. It's uses range from allowing the user to have a documented outline of a practice presentation which they can move forward with, to letting more spontaneous speakers change the order of things on the fly without having to worry that their powerpoint won't match.

The speech-to-text used IBM's Bluemix API, images were gathered from Getty Images and Wolfram Alpha, and slides were rendered using RevealJS. I worked primarily on the keyword logic and on making calls to the image APIs.
